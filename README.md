Many students fail or drop out due to late identification of academic difficulties.
This system predicts student risk levels using historical academic and demographic data.

Dataset
Source: Student Performance Dataset
Features include:
Gender, age, study time
Parental education
Past failures
Internal and final grades
Methodology
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Feature engineering and encoding
Train-test split
Model training using Random Forest
Model evaluation and feature importance analysis
Machine Learning Model
Random Forest Classifier
Output
Students are classified into:

Safe
At Risk
High Risk
